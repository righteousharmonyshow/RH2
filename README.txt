Righteous Harmony — Site Bundle (v5)

1) Upload ALL files + the `assets/` folder to your GitHub repo root.
2) Netlify auto‑deploys. If needed, trigger a manual redeploy.
3) Swap any image/PDF by replacing the file in `/assets` with the same name.
4) In `sizzle.html`, replace YOUTUBE_VIDEO_ID with your video ID.
5) To enable GA4, set window.GA_ID before analytics.js or edit the script with your ID.
